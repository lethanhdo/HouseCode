Cú pháp CSS

Bộ chọn trỏ đến phần tử HTML mà bạn muốn tạo kiểu.

Khối khai báo chứa một hoặc nhiều khai báo được phân tách bằng dấu chấm phẩy.

Mỗi khai báo bao gồm tên thuộc tính CSS và một giá trị, được phân tách bằng dấu hai chấm.

Một khai báo CSS luôn kết thúc bằng dấu chấm phẩy và các khối khai báo được bao quanh bởi các dấu ngoặc nhọn.

Tài liệu CSS đơn giản

Ví dụ

< DOCTYPE html >
< html >
< tittle > Page Tittle < /tittle >
< body >

< h1 > My First Heading < /h1 >
< p > My first paragraph. < /p >
< /body >
< /html >

Giải thích ví dụ

  • Các <!DOCTYPE html> tuyên bố xác định tài liệu này là HTML5
  • Phần <html> tử này là phần tử gốc của trang HTML
  • Phần <head> tử chứa thông tin meta về tài liệu
  • Phần <title> tử chỉ định một tiêu đề cho tài liệu
  • Phần <body> tử chứa nội dung trang hiển thị
  • Phần <h1> tử xác định một tiêu đề lớn
  • Phần <p>tử định nghĩa một đoạn
