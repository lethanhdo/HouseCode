HTML


Ngôn ngữ để xây dựng trang web

HTML là ngôn ngữ đánh dấu tiêu chuẩn cho các trang Web.
Với HTML, bạn có thể tạo Trang web của riêng mình.
HTML như là bộ khung của trang web.
Ví dụ: bạn có thể hiểu một cách đơn giản HTML là một viên kim cương thô...

CSS


Ngôn ngữ mô tả phong cách của một tài liệu HTML.

CSS là ngôn ngữ mô tả phong cách của một tài liệu HTML.
CSS mô tả cách các phần tử HTML sẽ được hiển thị.
Ví dụ: bạn có thể hiểu một cách đơn giản CSS làm viên kim cương thô kia có màu đỏ...

JavaScript


Ngôn ngữ lập trình của HTML và Web.

JavaScript là ngôn ngữ lập trình của HTML và Web.
Với JavaScript, bạn có thể lập trình hành vi của các trang web
JavaScript như là sự sống của trang web.
Ví dụ: bạn có thể hiểu một cách đơn giản JavaScript giúp viên kim cương của bạn có thể chuyển động ...

VÍ DỤ CODE HTML

< DOCTYPE html >
< html >
< tittle > Html Tutorial < /tittle >
< body >

< p > < /p >

< /body >
< /html >

Hãy nhập vào khoảng trống phía trên bất kỳ thứ gì.
